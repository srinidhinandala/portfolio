Core 4

Variables

Variables contain data types that will store information that can be changed. Sometimes, a user will input a value that will make the variable change

Loops

Loops are a sequence of commands that is repeated indefinitely. A long block of code with repeated instructions can be condensed with the use of loops.

Functions

Functions are steps of code that are versatile and can be used in multiple places. Instead of typing the same lines of code again, a function can be used for the same commands to work out.
